# User Guide

面向终端用户的中文文档。**本目录在 v1.0 之前主要为占位**，等核心功能稳定后再系统填充。

> 占位原因：v0.0.x 阶段 UI / 数据源 / 行为仍频繁迭代，提前写用户文档会持续过期。  
> 母法 §7.3 第 5 条把"中文 user-guide 完整"列入 v1.0 硬清单。

## 计划文件

> 以下为**占位计划**，对应文件在 v1.0 之前按触发版本陆续填充。占位期间不建实文件以避免无内容索引。

| 文件 | 内容 | 触发版本 |
|---|---|---|
| `getting-started.md` | 安装、首次登录、菜单栏含义 | v0.1.0 起补 |
| `faq.md` | 常见问题、限额解释、安装报错 | v0.2.0 起补 |
| `privacy.md` | 隐私模型、权限申请清单、数据流图、用户可审计步骤 | v0.2.x 引入 cookie/JSONL 时强制 |
| `screenshots/` | 截图素材，用于 user-guide 与 README | v1.0 前 |

## 写作约定

- 中文优先；技术术语保留英文
- 第一人称是用户，避免"系统"、"应用"等冷词
- 每个 how-to 都有可粘贴的命令或可点击的菜单路径
- 截图统一在 macOS Sonoma+ 浅色模式下生成（除非主题相关）
