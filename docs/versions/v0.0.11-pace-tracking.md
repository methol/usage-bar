# v0.0.11 — pace-tracking

## 主题

5h 窗口配速指示器三态（参考 CodexBar）：
- **On pace** 默认状态不显示
- **N% over pace · runs out in HhMm**（红）— 当前速率会在 reset 前用完
- **N% under pace**（绿）— 用量较慢

显示在 hero card 进度条下方；早期窗口（< 3% elapsed）静默；不引入 ML 等营销话术。

不做 7d pace、不做菜单栏 pace、不做通知（留后续）。
