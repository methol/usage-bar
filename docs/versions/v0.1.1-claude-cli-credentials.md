## 主题

- 读 `~/.claude/.credentials.json`
- 读 Keychain item `Claude Code-credentials`（如不可读则尝试 Security CLI 兜底，参考 CodexBar 的 "Avoid Keychain prompts"）

本版本同时引入 **`ClaudeUsageStrategy` 协议**作为后续 strategy chain 抽象层：
- `BuiltinOAuthStrategy`（现有逻辑）
- `ClaudeCLICredentialsStrategy`（本版本新增）

`UsageService` 内部循环调用 strategy chain，记录 attempts + errors，Debug 菜单露出。
