## 主题

WidgetKit 扩展。参考 CodexBar 的 widget 设计（砍掉 Switcher，因为我们 Claude-only）：

- **Usage widget**（small/medium/large）：当前 5h / 7d 使用率 + reset
- **History widget**（medium/large）：近 7 天使用率趋势图
- **Compact widget**（small）：today-cost / 30d-cost（依赖 v0.1.2 cost 扫描）

- App Group 容器写 `WidgetSnapshot` JSON，widget 读
