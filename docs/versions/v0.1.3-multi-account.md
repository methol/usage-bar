# v0.1.3 — multi-account

## 主题

让一份 app 同时管理多个 Claude OAuth 账号；popover 顶部下拉切换。承接 v0.1.1 ClaudeUsageStrategy 协议骨架与 v0.1.2 LocalCost 的多数据源演进。

精简范围：本 spec 仅做"账号 store + 迁移 + 切换器 + 添加入口"；per-account history 隔离 / Settings 删除 UI / merge icons 留 v0.1.4 / v0.2.x。
