# v0.0.9 — trend-arrows

## 主题

引入趋势箭头 ▲ / ▼ + 增量百分点（如 `▼ 2%`）。基于现有 30 天 `history.json` 计算近 6h 趋势，零额外存储。在 5h hero / 7d secondary 卡片 label 行内显示。

不动 history 数据模型、不动 OAuth / 数据层、不显示 ExtraUsage 与 Per-Model 趋势（数据语义不同）、不动菜单栏图标（v0.0.10 才做）。
