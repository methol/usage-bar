# v0.5.3 — observable-migration-release

## 变更概要

- 9 个类（ProviderRuntime / UsageService / ProviderCoordinator / UsageHistoryService / NotificationService / UsageStatsService / AppUpdater / LaunchAtLoginModel / MultiMenuBarLabel）从 `ObservableObject` + `@Published` 迁移到 Swift Observation framework 的 `@Observable`
- 删除 `RuntimeAggregator`（手动 Combine 聚合）
- 删除 `UsageService.runtimeAuthSync`（改 `isAuthenticated didSet`）
- `LaunchAtLoginModel` 拆出为独立文件
- 视图层全面改用 init 注入，删除 `@EnvironmentObject`
