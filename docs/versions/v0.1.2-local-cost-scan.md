# v0.1.2 — local-cost-scan

## 主题

扫本地 JSONL 日志算 per-model token + 30 天 USD 累积，作为 OAuth API 之外的数据深度补充。承接 v0.1.1 已落地的多数据源策略骨架。

含**隐私架构守护**：parser schema 主动不 decode `message.content` 字段，从结构上防止用户对话内容流入日志/缓存。
