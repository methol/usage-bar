# v0.2.11 — 统一后台 timer + Codex 菜单栏 glyph

## 主题

收尾两个 v0.2.10 留后续项：

- **`ProviderCoordinator` 真正统管所有 provider 的后台轮询 timer**（含 Claude）—— `UsageService` 退役自持 `Timer`，429 backoff 改成「这个时刻之前别 tick 我」的只读 hint，coordinator 的单个 timer 每 tick 跳过还在 backoff 窗口里的 provider。代价：backoff 解析度从「精确按 backoff 间隔」变成「round 到下一个 polling 间隔」（最坏多等一个周期 —— backoff 本就是别 hammer 服务器，无害）。
- **Codex 菜单栏专属 glyph** —— 取代 v0.2.10 临时用的 SF Symbol `terminal`，改成代码绘制的单色标记（不引入图片资源）。

**不含**：菜单栏在 Codex 作 menu-bar provider 时显示趋势箭头（仍 Claude-only，留后续）；给 Codex 做真·图片资源 logo（先用代码绘制的 glyph）；`UsageService` 的 OAuth/refresh/多账号逻辑改动（只动 timer/backoff 那块）。
