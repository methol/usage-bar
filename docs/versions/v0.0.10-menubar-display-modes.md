# v0.0.10 — menubar-display-modes

## 主题

菜单栏 3 种显示模式可切换（Settings → General → Menubar Display）：
- **icon**（默认，现状）：双窗口进度条图标
- **percent**：文本 `5h 42%`
- **percent + trend**：文本 `5h 42% ▼5`（复用 v0.0.9 趋势）

`$/天` / token 等模式留给 v0.1.2 数据源 ready 后再加，本版本不引入 dead UI 选项。
固定显示 5h 窗口（7d 切换 / 双窗口模式留后续）。
