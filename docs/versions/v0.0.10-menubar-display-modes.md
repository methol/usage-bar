## 主题

菜单栏多显示模式：`icon` / `percent` / `percent + 趋势` / `$/天 + 趋势` 四种可切换。
当前 `MenuBarIconRenderer` 仅支持图标模式。本版本添加文本渲染分支与 Settings 选项。

`$/天` 模式所需的 cost 数据要等 v0.1.2 本地 JSONL 扫描；本版本仅先提供前两种模式 + 趋势，`$` 模式 UI 显示但数据空（"待 v0.1.2"提示）。
