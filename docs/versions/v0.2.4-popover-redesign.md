# v0.2.4 — Popover Redesign

## 主题

Popover 大改版，三块改动：

1. **多 provider tab 入口**：popover 顶部加一排 tab（Claude / Codex / Cursor / Copilot / Gemini），目前只有 Claude 可用，其余显示"敬请期待"占位面板 —— 为后续对接 Codex 搭好 UI 外壳（[ADR 0005](../adr/0005-reopen-multi-provider-direction.md) supersede 了原来的"只做 Claude" ADR 0002）。
2. **卡片化视觉重做**：5h / 7d 用量改成圆角卡片（时钟 / 日历图标 + "Resets in: 2h 46m at 11:44 PM" + "Pace: safe / fast" 标签），popover 背景换成柔和渐变；放弃 v0.0.8 的 56pt 大字号 hero，两卡等权更紧凑。
3. **折线图 pace 面积**：在现有 5h（蓝）/ 7d（橙）两条折线下方叠两片极浅的"配速"面积 —— 一眼看出此刻你比"匀速用完额度"的理想线快了还是慢了。原折线和悬停明细不变。
