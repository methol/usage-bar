# v0.3.1 — swiftui-hygiene

## 主题

v0.3.0 Provider 自主管理 merge 后的 SwiftUI 现代化 audit 落地版本之一（共三个版本：v0.3.1 hygiene → v0.4.0 view-layer → v0.5.0 @Observable）。

本版本只收 audit 中**风险最低**的一组：3 个真实 high bug（崩溃路径 + 性能 + 可访问性）、5 条 low hygiene、2 处死代码下线。**不**触动 SettingsView Binding / PopoverView ViewBuilder（v0.4.0）、**不**触动 ObservableObject → @Observable 迁移（v0.5.0）。
