# v0.0.7 — docs-governance

## 主题

立项后第一个独立编号版本。**不含代码变更**，仅建立文档治理框架，作为后续所有功能升级（v0.0.8 ~ v1.0.0）的母法。

如果不先立这一版，后续每个功能版本都将面临：
- AI 不知道用什么模板、放哪里
- 决策没有 trace，下个 AI 会话无法接续
- 没有 review 机制，AI 自审自批
- 与上游 Blimp-Labs 的版本号 / URL 冲突
