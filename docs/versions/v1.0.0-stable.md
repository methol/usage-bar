## 主题

🚀 **稳定可用**。母法 §7.3 列出 14 条硬清单，任一项不满足停在 v0.x：

1. 主功能零 known critical bug 满 30 天
2. Apple Developer ID 公证（用户安装无右键 Open 提示）
3. Sparkle 自动更新经历过 ≥3 个版本周期且零事故
4. 至少 2 条 Claude 数据源路径可用（OAuth + 一种回退）
5. 中文 user-guide 完整（getting-started + faq + privacy + screenshots）
6. CHANGELOG.md 自 v0.0.7 起完整中文记录
7. 核心算法（pace / trend / cost-scan）单元测试 line coverage ≥80%
8. `docs/runbooks/release.md` Runs log 表 ≥3 次成功 run
9. 性能基线：idle CPU < 1%、内存 < 80MB、cold start < 800ms
10. 能源：Activity Monitor → Energy 不出现 "Significant Energy Impact"
11. 隐私 audit：完整权限申请清单 + 数据流图 + 用户可审计步骤
12. a11y：VoiceOver 朗读 menu bar 图标 / popover 数字 / 进度条
13. 本地化决策：明确表态 v1.0 是否含中文 UI
14. 崩溃监控：明确表态是否接 crash reporting + 方案
