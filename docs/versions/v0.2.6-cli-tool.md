## 主题

`claude-usage-bar` CLI 工具。复用 strategy chain 抽象层（v0.1.1 引入），提供脚本化使用数据访问：

- `claude-usage-bar usage --format json --pretty`
- `claude-usage-bar cost --format json --refresh`
- `claude-usage-bar cache clear`
- `claude-usage-bar config validate / dump`

实施要点：
- 新加 SwiftPM target `ClaudeUsageBarCLI`
- 与 app 共享 `~/.config/claude-usage-bar/credentials.json`（多账号）与本地 cost cache
- 通过 app 内 Settings → Advanced → Install CLI 创建 symlink 到 `/usr/local/bin` / `/opt/homebrew/bin`
- 可选：发 macOS / Linux tarball，但 Linux 仅限不依赖 macOS 框架的子命令（如 cost 扫描）
