# v0.2.14 — litellm-pricing

## 主题

模型「估算费用」的价格表不再手写维护，改为打包 [LiteLLM](https://github.com/BerriAI/litellm) 的社区价格快照（`model_prices_and_context_window.json`），并在运行期每 3 小时后台刷新一次。再加一层「模型别名 → 有价模型」的逐级回退查表，让 codex CLI 的内部别名（`gpt-5.3-codex` / `gpt-5.1-codex-max` / `gpt-5.4-mini` / `gpt-5.4-xhigh` …）也能匹配到价格——修掉 Codex tab 频繁出现的「含 N 条未知模型调用（价格表过时？）」。

用户感知：Codex / Claude 估算费用更准、「未知模型」提示基本消失；价格随上游自动更新，不再依赖发版。
